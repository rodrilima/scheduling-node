## Getting Started

### Prerequisites

To run this project, you'll only need to have the npm installed.

### Installing

Cloning the Repository

```
$ git clone https://github.com/rodrilima/scheduling-node.git

$ cd scheduling-node

$ npm install
```

### Running

```
$ npm start
```
